Getting Started

Let's get started!

Requirements​

Before installing Jenova Runtime on your Game Engine you need to check for some requirements :

  • A Compatible Godot Engine 4.2+ Fork

    Information

    All Godot forks are capable of running Jenova Runtime but Hot-Reloading is only supported on following distros :

    • Godot Jenova Edition
    • Blazium Engine
    • Redot Engine

  • At the moment Jenova is only supported on 64-bit Windows Operating System. You need to have a standard Windows 7 to 11.

    Note

    Linux is supported but currently not functional due to unfinished Metadata Extractor.

  • Visual Studio 2017-2022 (Optional)

Installation​

For Installing Jenova Runtime you can use Jenova Installer or do it manually.

Automatic​

You can download the installer from here, Simply run installer and select your Godot project path. After installer closed, Open Godot.

  1. Run installer, Select path of your Godot project you want to install Jenova Runtime in.
  1. Click on Install Jenova Framework and wait until installer finish.
Note

The official installer is only available for Windows. Linux users will need to build it from source.

Manual​

  1. Head to Release Page of Jenova repository, Download latest framework release, It looks like this :

    Jenova.Framework.<Version>.<Release Stage>.Build.<Build Number>.Win64.zip

  2. After you downloaded the zip file extract it to your project directory, Jenova Framework must be placed at res://Jenova and using any other path will lead to malfunctioning.

  1. Run Godot once after extraction, Close Godot and open it again. (See Known Bugs)
